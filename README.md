## Local Development

```sh
nvm install            # uses .nvmrc (Node 14.16.0)
yarn install
npm start              # http://localhost:8000
```

Production build:

```sh
npm run build
npm run serve
```

## Helper Scripts

- `scripts/screenshot-sites.sh` — regenerate live screenshots for all featured projects via headless Chrome.
- `scripts/gen-covers.js` — generate themed monogram cover fallbacks (sharp).
- `scripts/gen-logo.js` — regenerate the K monogram logo + favicons.
